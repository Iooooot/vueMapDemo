# map_demo

基于VueCli搭建的中国人口统计地图

## 项目描述

### 项目说明

- map_demo是一个利用VueCli搭建的vue项目用于人口统计，使用Echarts绘制的中国地图
- 主要涉及技术有：VueJs、VueRouter、Echarts

### 效果展示

![](https://blogpic-1305209282.cos.ap-chengdu.myqcloud.com/img/QQ%E5%BD%95%E5%B1%8F20230424164550.gif)

### 功能

- 地图的放大/缩小
- 地图回退/下钻
- 人口数据统计

## 项目运行

1. `npm install`
2. `npm run serve`
